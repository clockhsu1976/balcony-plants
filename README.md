# 陽台植物分布圖 🪴

記錄陽台 23 株植物的位置、喜水性、耐曬程度與介質肥料的互動式圖鑑。純前端單一 HTML 檔，可離線開啟、可拖曳編輯，版面存在瀏覽器本機。

## 功能
- 分布圖：依實際陽台俯視位置擺放植物，分三區（洗手台區、冷氣外機區、陽台主區）。
- 喜水性：編號顏色代表澆水需求 — 🟠 耐旱、🟢 適中、🔵 喜水。
- 耐曬程度：植物右上角小圖示 — ☀️ 耐曬、⛅ 半日照、🌿 喜陰；主區右上角有午後強光區提示。
- 建議擺放區 1：依耐曬＋喜水性在各區內重新排列。
- 建議擺放區 2（跨區）：含跨區搬移建議，換區植物以紅色虛線框 + ↪ 標記。
- 介質＋肥料：分群速查卡 + 各株完整附表（介質配方、肥料、施肥頻率）。

## 操作
- 上方「🗺️ 分布圖 / 🪴 介質＋肥料」切換頁面。
- 編輯模式下：拖區塊空白處整塊移動、拖植物在區內移動、拖區塊右下角改大小、拖大框框底部中央把手改整體高度。
- 按「💾 儲存版面」記住調整（存該裝置瀏覽器，不跨裝置同步）。
